# Multilingual Pronunciation Practice

## Overview

This is a web app that tests dictation in multiple possible languages. It presents a sentence and prompts you to record your pronunciation of the sentence through your device's microphone. An ASR model is used to transcribe the audio and then show the you what you actually said. A TTS model produces an audio file of the sentence, giving you an example of how the sentence should be pronounced.

## Supported Languages

- English
- Vietnamese

## Requirements

Requires Python 3.12
